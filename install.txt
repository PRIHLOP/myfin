Установка:
	1. Переименуйте файл src/app/example_config.php в config.php
	2. Отредактируйте config.php, следуя подсказкам внутри
	3. Загрузите содержимое директории src в папку htdocs на сервере
	
Обновление:
	1. Скопируйте содиржимое src в htdocs с заменой при совпадении имен
	
При обновлении с 1.0.25 (и младше) до 1.0.28 (и старше):
	1. Скопируйте содиржимое src в htdocs с заменой при совпадении имен
	2. Скопируйте недостающие опции из example_config.php в свой config.php
	3. Загрузите на сервер файл updateDb.php и перейдите по ссылке http://ваш.сайт/updateDb.php
	4. Удалите updateDb.php с сервера
